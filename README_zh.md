## 中文版

### 简介
checkARB 是一个自解压 Shell 脚本，用于提取并检查 Android 设备 `xbl_config` 分区中存储的反回滚版本信息。它利用内置的 `arb_inspector` 工具解析分区镜像，并通过直观的颜色高亮显示结果。脚本遵循 POSIX 标准，可在 `mksh`、`ash` 等常见 Android Shell 中稳定运行。

### 功能特点
- **自解压**：包含 `arb_inspector` 的 ZIP 归档附加在脚本末尾；运行时自动解压到临时目录，无需手动准备工具。
- **双源检查**：可选择直接从**本地分区**提取 `xbl_config`，或检查用户提供的**外部镜像文件**。
- **反回滚检测**：通过专用工具解析 `Anti-Rollback Version`。数值 > 0 时以红色高亮显示（“已启用反回滚”），数值为 0 时以绿色显示（“无反回滚”）。
- **联发科天玑警告**：自动检测联发科天玑芯片，若存在则显示黄色警告，提示 ARB 值可能存储在硬件中，结果可能不可靠。
- **备用 busybox**：当系统命令缺失时，递归搜索 `/data/adb` 寻找可执行的 `busybox`，提升不同环境下的兼容性。
- **POSIX 兼容**：使用纯 POSIX Shell 编写，在 `mksh` 和 `ash` 上完美运行，同时检测到 `bash` 时会退出并报错。
- **清理保证**：无论成功或失败，脚本通过 `trap` 在退出时自动删除工作目录（`/data/local/tmp/checkarb`），不留残留。
- **哈希验证**：内嵌的 `bin.zip` 会通过 SHA256 校验和进行验证，防止篡改或损坏。
- **用户友好界面**：纯文本菜单引导用户选择源、确认提取和输入外部路径，无需 `dialog` 等额外工具。
